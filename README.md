### 功能简介

该脚本用于分析微信聊天记录，生成年度报告。它能够处理单聊和群聊数据，提取关键信息并生成可视化报告。该程序支持输入csv格式的聊天记录文件，并输出HTML格式的年度报告。

### 使用方法

1. 克隆或下载本仓库到本地。
2. 确保你的系统已安装Python 3.x。
3. 安装所需的Python库，可以使用以下命令：
   ```
   pip install -r requirements.txt
   ```
4. 准备你的聊天记录CSV文件，并将其放在项目根目录下。如果你不知道如何导出微信聊天记录，可以参考后文《如何导出微信聊天记录》。
5. 运行脚本生成报告：
   ```
   python wechat_analysis.py
   ```
6. 生成的年度报告将保存为`WeChat_Report_YYYY_Final.html`。
7. 打开生成的HTML文件，查看你的微信年度报告。

### 如何导出微信聊天记录
1. 检查你的**电脑微信**版本，确保它不是4打头的版本。如果是，请降级到3.9.12版本（该版本可以在网上找到安装包）。
2. 通过微信电脑版登录你的账号。
3. 进入“设置”->“聊天”->“聊天记录迁移”，请将全部聊天记录迁移到电脑端（这个过程可能需要一些时间，取决于你的聊天记录大小，本人花了大约4个小时）。迁移在换手机时也可以使用，这里不赘述具体方法，可以参考微信官方帮助文档。
4. 迁移完成后，下载Memotrace。（该工具本来可以在GitHub上找到，但是似乎被下架了，我传了一个备份在.\MemeTrace里面）
5. 打开Memotrace.exe，点击“获取信息”，等待自动分析完成。
6. 点击“设置微信路径”，检查是否是正确的路径（一般会自动识别）。
7. 点击“解析数据”，等待完成。
8. 解析完成后，点击最上方“数据”，选择“导出聊天记录（全部）”，选择保存路径，保存为CSV格式。
9. 将导出的CSV文件放在项目根目录下。
